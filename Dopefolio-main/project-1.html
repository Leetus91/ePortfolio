E-gaming Form & SQL Database

George Campanis, in our 1st year course for SQL, assigned us a Group Project to create a Database.

Code Link

Project Overview

The three of us started out by selecting an E-Gaming Form, so that we could link it via OBDC connection to SQL, and show an ERD relationship diagram via Microsoft Access.

16 teams, 16 coaches and 5 players per team were added into our Database. We included unique emails, gamertags, and personal information to locate members. Chris created an ERD, I created the SQL Database with Jon, along with MS Access form and together we created a complete project, which George gave us a 98% final marking on. Over-all it was an excellent learning experience on the fundamentals of SQL.
